## Git 常用命令
### <span style="color:red;">工程准备</span>
工程克隆——**git clone**

### <span style="color:red;">查看工作区</span>
查看工作区的修改内容——**git diff**
查看工作区文件状态——**git status**

### <span style="color:red;">文件修改后提交推送</span>
新增/删除/移动文件到暂存区——**git add/ git rm/ git mv**
